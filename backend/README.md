# PyTorch Deep Learning Studio Backend

This directory contains the Python FastAPI backend for the PyTorch Deep Learning Studio web application.

## Setup

```bash
pip install -r requirements.txt
```

## Running

```bash
python main.py
```

The server will start on http://localhost:8000

## API Documentation

Once running, visit:
- Swagger UI: http://localhost:8000/docs
- ReDoc: http://localhost:8000/redoc
